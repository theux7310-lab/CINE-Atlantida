# CINE-Atlantida
O Cine Atlântida é um projeto de desenvolvimento web que tem como objetivo criar um sistema completo para o gerenciamento de um cinema. Atualmente, o projeto encontra-se em desenvolvimento e esta versão representa a primeira etapa de sua implementação.

Nesta fase, foi desenvolvido o módulo de CRUD (Create, Read, Update e Delete), responsável pelo cadastro, consulta, edição e exclusão de clientes e filmes. O sistema foi construído utilizando HTML5, CSS3 e JavaScript, com persistência de dados através do LocalStorage, permitindo o armazenamento das informações diretamente no navegador.

O projeto foi planejado de forma modular para facilitar futuras expansões, visando evoluir de uma aplicação simples para um sistema mais completo e robusto.

# Funcionalidades Implementadas
- Cadastro de clientes e filmes;
- Consulta de registros;
- Edição de informações;
- Exclusão de registros;
- Armazenamento local com LocalStorage;
- Interface simples, organizada e responsiva.

# Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage

# Status do Projeto

Em desenvolvimento.

Até o momento, apenas o módulo de gerenciamento de clientes e filmes (CRUD) foi concluído. Novas funcionalidades serão adicionadas nas próximas versões para tornar o sistema mais completo.

# Funcionalidades planejadas

- Sistema de login e autenticação;
- Integração com banco de dados (MySQL ou PostgreSQL);
- Backend utilizando Node.js, Java ou PHP;
- Pesquisa e filtros avançados;
- Cadastro de sessões e horários;
- Reserva e venda de ingressos;
- Gerenciamento de clientes;
- Painel administrativo com estatísticas;
- Upload de pôsteres dos filmes.

O Cine Atlântida é um projeto voltado ao aprendizado e à prática dos conceitos de desenvolvimento web, servindo como base para a construção de um sistema de gerenciamento de cinemas cada vez mais completo.
